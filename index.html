Project at a Glance

  • Innovative Security Solution: Utilizes a pressure mat as a trigger for home security alerts.
  • Real-Time Email Alerts: Sends immediate email notifications when the pressure mat is activated (Stepped on)
  • DIY and Cost-Effective: Built using a Raspberry Pi, Python code and a standard pressure mat, offering a budget-friendly security solution
  • Easy Integration: Seamlessly integrates into existing home network
  • Continuous Monitoring: ideal for continuous monitoring using the infinite python while loop
  • Versatile Application: Suitable for various settings, including homes, offices, or garages
  • Enhanced Safety: Provides an additional layer of security for peace of mind
